# Sample API with Database

A Flask REST API with SQLAlchemy and Alembic for database migrations.

## Quick Start

See `runbook.ipynb` for step-by-step instructions to build and run this project.

## Stack

- Flask + flask-restx for REST API with Swagger UI
- SQLAlchemy for ORM
- Alembic for database migrations
- SQLite (default) or PostgreSQL
